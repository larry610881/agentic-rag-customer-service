# 成本比對分析

> **定價基準日**：2026-03-05
> **匯率**：1 USD = 32.2 TWD
> **部署區域**：GCP asia-east1（台灣彰化）
> **LLM 實測基準**：GPT-5.1，3 輪電商客服 RAG 對話

---

## 一、成本結構拆解

自建方案的成本由兩部分組成：

| 成本類型 | 說明 | 特性 |
|---------|------|------|
| **基礎設施（固定成本）** | GCP 雲端服務：運算、資料庫、向量搜尋 | 不隨對話量變動，隨租戶數升級 |
| **LLM API（變動成本）** | 呼叫 AI 模型的費用 | 按用量計費，用多少算多少 |

LumineAI 把這兩者打包成一個 token 包（18M / 50,000 TWD），無法拆開看成本結構。

---

## 二、基礎設施成本（GCP）

### 三級距設備規格

系統依租戶數量分三個級距，基礎設施費用隨規模遞增：

| 級距 | 租戶數 | 月對話上限 | 月成本（無 HA） | 月成本（含 HA） |
|------|-------|----------|---------------|---------------|
| Tier 1 起步 | 1-2 家 | ≤ 5,000 | **$70/月**（2,254 TWD） | 不需要 HA |
| Tier 2 成長 | 3-10 家 | 5K-30K | **$164/月**（5,281 TWD） | **$336/月**（10,819 TWD） |
| Tier 3 規模化 | 10-30 家 | 30K-100K | **$277/月**（8,919 TWD） | **$553/月**（17,807 TWD） |

### 基礎設施組成（以 Tier 1 為例）

| 服務 | 用途 | 月成本 |
|------|------|--------|
| Cloud Run | 後端 FastAPI 應用 | $18 |
| Cloud SQL | PostgreSQL 資料庫 | $10 |
| GCE VM | Qdrant 向量搜尋引擎 | $19 |
| Cloud Storage | 前端靜態檔託管 | $1 |
| 其他 | 網域、IP、Registry | $6 |
| 合計 | | **$70/月** |

> 不需要 Redis — 系統內建 InMemoryCache，Demo 和初期可省 $24/月。
> 不需要 Load Balancer — Cloud Run 自帶 TLS URL，初期可省 $19/月。

## 三、LLM API 成本

| 指標 | V1 原始 | V3 優化（推薦） |
|------|--------|--------------|
| 每次對話 Input tokens | 12,015 | 4,931 |
| 每次對話 Output tokens | 566 | 384 |
| 每次對話 Embedding tokens | 300 | 300 |
| **每次對話合計** | **12,881** | **5,615** |
| **月合計（1,000 次對話）** | **12.9M** | **5.6M** |

## 四、完整月成本：自建 vs LumineAI

### 以 18M Tokens 為基準的總成本比較

> 自建方案含基礎設施 + LLM API，LumineAI 為全包價。
> 自建使用 V3 優化配置。

| 方案 | LLM 費用 | 基礎設施 | 18M 總成本 (TWD) | 可服務對話數 | 每次對話成本 |
|------|---------|---------|-----------------|-----------|-----------|
| **LumineAI** | 50,000 TWD（全包） | 含在內 | **50,000** | 依方案而定 | **35.8 TWD** |
| 自建 V3 GPT-5.1 | 1,033 | 7,233（3.21 月） | **8,266** | 3,206 次 | **2.58 TWD** |
| 自建 V3 DeepSeek | 160 | 7,233（3.21 月） | **7,393** | 3,206 次 | **2.31 TWD** |

> **同樣 18M tokens，LumineAI 收 50,000 TWD，自建含基礎設施最多 8,266 TWD — 便宜 6 倍。**
> 每次對話成本：LumineAI 35.8 TWD vs 自建 2.3-2.6 TWD — **便宜 14 倍**。

### 每月營運成本對照（每租戶 1,000 次對話）

| 方案 | 月成本 | 年成本 |
|------|--------|--------|
| **LumineAI**（按比例換算） | ~35,833 TWD | ~430,000 TWD |

## 七、關鍵結論

1. **自建 vs LumineAI 總成本差距 6 倍以上**（含基礎設施的完整比較）
2. **每次對話成本差距 14 倍**：LumineAI 35.8 TWD vs 自建 2.3-2.6 TWD
3. **推薦定價帶 = LumineAI 的 40-50%**：客戶省一半以上，我方毛利率 25-53%
4. **年淨利約 15 萬 TWD**（收半價 + 全 HA + GPT-5.1 最保守估算）
5. **成本可進一步壓低**：改用 DeepSeek V3.2 或 Gemini 2.5 Flash，LLM 成本再降 75-85%

> **注意**：以上毛利率未扣除開發人力（一次性）和維運人力（持續），需依團隊規模另行評估。LumineAI 的 50,000 TWD 為全包價（含開發 + 維運 + 技術支援）。
