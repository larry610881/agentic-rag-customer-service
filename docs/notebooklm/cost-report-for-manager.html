測試機環境需求與成本評估

部長好，以下是測試機環境需求與成本評估：

一、GCP 基礎設施（必要）

服務用途月費
Cloud Run後端 API 服務~$0-15
Cloud SQL PostgreSQL資料庫（含向量搜尋 pgvector）~$50
Memorystore Redis快取 / Session / Rate Limit~$35
VPC + Registry + 其他網路連線 + 容器儲存~$8.5
基礎設施小計~$93-108（約 3,000-3,500 TWD）

二、前端部署方案

前端是靜態網頁（SPA + 嵌入式 Widget），有兩個方案：

方案月費說明
Firebase Hosting（建議）$0-5Google 旗下產品，同一個 GCP 帳號 / 同一張帳單。內建全球 CDN + 自動 SSL + 自訂域名，5 分鐘可設定完成
GCS + Cloud CDN + Load Balancer~$20-30需要額外設定 GCS → CDN → 負載均衡器 → 域名，其中負載均衡器固定月費 $18。功能相同但多花 $20/月，infra 設定都可以幫忙處理

現階段建議用 Firebase Hosting，年省約 $240-300。
